### 👋 About Me

I'm a backend WordPress developer who builds **custom, production-grade themes** — no page-builder bloat, no unnecessary abstraction. Just clean PHP, ACF Pro, and Gutenberg block systems built to last.

I work closely with senior devs' established conventions and architecture, following consistent patterns across every project so the codebase stays predictable for the next person who opens it.

**Right now:** building a full custom theme for a 10-member Nepali band — CPTs, taxonomies, Fancybox video popups, Splide sliders, and an ACF Options Page powering global content.
